# mini гайд по приложению

## Как скачать FFmpeg

1. Перейдите на официальный сайт: https://ffmpeg.org
2. Нажмите на "Download" и выберите сборку для вашей ОС.
3. Для Windows можно скачать сборку с одного из зеркал, например: https://www.gyan.dev/ffmpeg/builds/ или https://www.gyan.dev/ffmpeg/builds/.
4. Скачайте архив ZIP с последней стабильной версией.
5. Распакуйте архив в удобную папку.

## Куда ставить FFmpeg

1. Создайте папку, например: `C:\ffmpeg` или `C:\Program Files\ffmpeg`.
2. Переместите содержимое распакованного архива, чтобы внутри папки `ffmpeg` была папка `bin`, содержащая `ffmpeg.exe`.
3. Пример структуры:
   - `C:\ffmpeg\bin\ffmpeg.exe`
   - `C:\ffmpeg\bin\ffprobe.exe`

## Добавление в PATH (рекомендуется)

1. Откройте "Параметры" → "Система" → "О программе" → "Дополнительные параметры системы".
2. Нажмите "Переменные среды".
3. В разделе "Системные переменные" выберите `Path` и нажмите "Изменить".
4. Добавьте новый путь: `C:\ffmpeg\bin`.
5. Сохраните изменения и закройте окна.
6. Проверьте в командной строке командой:
   ```
   ffmpeg -version
   ```

## Как пользоваться в приложении

1. Убедитесь, что `ffmpeg.exe` доступен в командной строке.
2. Запустите приложение и следуйте инструкциям по конвертации.
3. Если приложение не находит FFmpeg, проверьте путь к `ffmpeg\bin` и переменную PATH.
